# Game Of Life

## To Run

Clone from github.

`git clone https://github.com/cowboy8625/GameOfLife.git`

Change Folders

`cd GameOfLife`

Compile to release

`cargo build --release`

Run Release

`cargo run --release`

## Controls

* R -> Random spawning,
* S -> Start and Stop,
* C -> Clear Screen,
* Left Click -> Place Cells,
* Up -> Speed Up,
* Down -> Speed Down,
* ESC -> Exit
